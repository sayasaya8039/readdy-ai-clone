コードを日本語で分かりやすく説明します。

## 実行内容

### 1. コード読み込み
- 指定されたファイル/関数を読み込み
- 関連するファイル（インポート先等）も確認

### 2. 分析
- 全体の目的・役割
- 処理の流れ
- 重要なロジック
- 使用しているライブラリ/パターン

### 3. 説明出力
- 概要（1-2文）
- 詳細な処理フロー
- 注意点・ポイント
- 改善提案（オプション）

## 説明レベル

| レベル | 対象 | 内容 |
|-------|------|------|
| 概要 | 初心者 | 何をするコードか |
| 標準 | 中級者 | 処理フロー、ロジック |
| 詳細 | 上級者 | 設計意図、最適化ポイント |

## 出力形式

```markdown
## 概要
このコードは〇〇を行います。

## 処理フロー
1. まず△△を取得します
2. 次に□□を処理します
3. 最後に◇◇を返します

## 重要なポイント
- ポイント1
- ポイント2

## 使用例
\`\`\`typescript
// 使用例
\`\`\`

## 関連ファイル
- `path/to/related.ts`
```

## オプション

| オプション | 説明 |
|-----------|------|
| `--brief` | 簡潔な説明のみ |
| `--detail` | 詳細な説明（行ごと） |
| `--diagram` | 処理フロー図も生成 |
| `--improve` | 改善提案も含める |

## 使用例

```bash
# ファイル全体を説明
/explain src/utils/auth.ts

# 特定の関数を説明
/explain src/utils/auth.ts:validateToken

# 詳細モード
/explain src/components/Modal.tsx --detail
```

$ARGUMENTS でファイルパスまたは関数名を指定。
