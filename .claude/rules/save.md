現在の作業内容とPlansをMemory MCPに保存します。

## 実行内容

1. **作業内容の整理**
   - 完了したタスク
   - 作成/変更したファイル
   - 重要な決定事項

2. **Memory保存**
   - プロジェクト情報を更新
   - 作業進捗を記録
   - 次回TODOを保存

3. **確認**
   - 保存内容を表示
   - 次のアクション提案

## 保存項目

| カテゴリ | 内容 |
|---------|------|
| プロジェクト | 名前、パス、技術スタック |
| 進捗 | 完了タスク、残タスク |
| 決定事項 | 技術選定、設計方針 |
| エラー | 発生したエラーと解決策 |
| TODO | 次回セッションでやること |

## 使用タイミング

- セッション終了前
- 重要な決定後
- エラー解決後
- コンテキスト圧縮前

$ARGUMENTS で保存するメモを追加可能。
