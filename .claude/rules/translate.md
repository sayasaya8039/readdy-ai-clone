日英翻訳を実行します。

## 実行内容

1. **言語検出**
   - 入力テキストの言語を自動判定
   - 日本語→英語 or 英語→日本語

2. **翻訳実行**
   - 自然な翻訳
   - 技術用語の適切な変換

3. **結果出力**
   - 翻訳結果を表示
   - ファイル指定時は上書きまたは新規作成

## 対応形式

| 形式 | 対応 |
|------|------|
| テキスト | ✅ |
| Markdownファイル | ✅ |
| コメント | ✅ |
| README | ✅ |

## オプション

- : 日本語に翻訳
- : 英語に翻訳
- : ファイル全体を翻訳

$ARGUMENTS で翻訳対象を指定。

## 技術用語

技術用語は適切に変換:
- コンポーネント ↔ component
- 依存関係 ↔ dependencies
- ビルド ↔ build
