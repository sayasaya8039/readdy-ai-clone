# Build Your Own X 活用ルール

## 概要

[build-your-own-x](https://github.com/codecrafters-io/build-your-own-x) は350k+スターの人気リポジトリ。
「自分で作れないものは理解していない」の理念に基づき、既存技術を一から実装するチュートリアル集。

## 参照すべき場面

| 場面 | アクション |
|------|-----------|
| **技術の深い理解が必要** | 該当技術のチュートリアルを参照 |
| **学習プロジェクト作成** | build-your-own-xから選定 |
| **面接準備・スキルアップ** | 基礎技術の実装練習 |
| **新言語学習** | 既知の技術を新言語で再実装 |

## 言語別チュートリアル

多くのチュートリアルが複数言語で提供：
- **C/C++**: 低レベル実装、OS、エミュレータ
- **Python**: ML、スクリプト系
- **Rust**: システムプログラミング
- **Go**: ネットワーク、CLI
- **JavaScript/TypeScript**: Web系

## 活用方法

1. **学びたい技術を選択**
2. **得意な言語のチュートリアルを選ぶ**
3. **段階的に実装を進める**
4. **完成後、別言語で再実装**（応用）
