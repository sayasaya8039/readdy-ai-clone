API設計とテストを支援します。

## 実行内容

1. **API設計**
   - RESTful設計
   - エンドポイント定義
   - スキーマ作成

2. **テスト実行**
   - リクエスト送信
   - レスポンス検証
   - エラーケース確認

3. **ドキュメント生成**
   - OpenAPI仕様書
   - 使用例

## 対応形式

| 形式 | 対応 |
|------|------|
| REST | ✅ |
| GraphQL | ✅ |
| tRPC | ✅ |

## RESTful設計原則

| メソッド | 用途 |
|---------|------|
| GET | リソース取得 |
| POST | リソース作成 |
| PUT | リソース更新（全体） |
| PATCH | リソース更新（部分） |
| DELETE | リソース削除 |
